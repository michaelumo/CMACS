# CMACS
Completion Monitor for Advanced Course Students

## Excelファイルについて

- ファイル名: cmacs_data.xlsx
- シート"0": コースに関する情報
- シート"4": 4年の科目
- シート"5": 5年の科目
- シート"s1": 専攻科1年の科目
- シート"s2": s2年の科目

- "name": 科目名
- "url": シラバスへのurl(相対バス)
- "condition": 選択又は必須
- "pt": 単位
- "jabee": jabee修了要件の項目A〜D
- "edu": 教育プログラム修了要件の項目1〜6
- "gpa": グレードポイントS,A〜D
- "spec": 人文科学・社会学をA, 数学・自然科学・情報技術をB 〜 専門工学の知識・能力 b,c,dをI
